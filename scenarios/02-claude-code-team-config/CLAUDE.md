# Project: FullStack App

You are working on a Python/FastAPI backend + React/TypeScript frontend monorepo.

## Team Norms
- All code must pass linting before commit
- Never modify database migrations without explicit user instruction
- Ask before deleting any file; propose moves instead
- Prefer editing existing files over creating new ones

## Stack
- Backend: Python 3.12, FastAPI, SQLAlchemy, Alembic, pytest
- Frontend: React 18, TypeScript, Vite, Tailwind CSS, Vitest
- Infrastructure: Docker, GitHub Actions

## Commit Style
Use conventional commits: `feat:`, `fix:`, `refactor:`, `test:`, `docs:`

## Path-Specific Rules
- `backend/` → see .claude/rules/backend.md
- `frontend/` → see .claude/rules/frontend.md
